## 🚨 NOTICE: This project has been deprecated

### What was Auctologue?

Auctologue was a boutique auction cataloging service that combined AI-driven automation with expert human refinement to create clear, accurate, and professional auction listings. The service was designed to streamline auction cataloging processes for auction houses and estate sellers.
